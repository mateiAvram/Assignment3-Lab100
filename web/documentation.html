Hava API Documentation

Basic Funcitonality:

Our RESTful API requests include the following modules:

  • /phone
  • /phones
  • /post
  • /update
  • /delete
  • /reset

GET /phone Request

URI Endpoint: 'http://localhost:8080/hava/phone?id=' + id

Error Codes: OK: 200, Not Found: 404, Bad Request: 400, Internal Err: 500

HTTP response MIME type: application/json

This API returns a single JSON object based on its id from the database. The object contains the following attributes in this order: id, brand, model, os, image and screensize.

GET /phones Request

URI Endpoint: 'http://localhost:8080/hava/phones'

Error Codes: OK: 200, Bad Request: 400

HTTP response MIME type: application/json

This API returns a vector containing JSON objects. Each element in the vector is a JSON object with the same attributes as a single phone entity.

POST /post Request

URI Endpoint: 'http://localhost:8080/hava/post'

Error Codes: Created: 201, Bad Request: 400, Internal Err: 500

HTTP response MIME type: text/plain

This API adds a new line in the database, based on the form in the webpage. As a front-end dev, you would need to create a JSON object in JS that contains the basic attributes in the specific order, then do the AJAX call which sends the object to the server and to the database next.

PUT /update Request

URI Endpoint: 'http://localhost:8080/hava/update'

Error Codes: No Content: 204, Not Found: 404, Bad Request: 400, Internal Err: 500

HTTP response MIME type: text/xml

This API changes an already existing line in the database based on the element's id. As a front-end dev, you would need to create a JSON object in JS that contains the basic attributes in the specific order, BUT make sure that the first attribute is 'id' with the appropriate value. Next, do the AJAX call which handles the connection to the server and database.

DELETE /delete Request

URI Endpoint: 'http://localhost:8080/hava/delete?id=' + id

Error Codes: No Content: 204, Not Found: 404, Bad Request: 400, Internal Err: 500

HTTP response MIME type: text/xml

This API deletes a line from the SQL database based on a specific id. Integrating this into your webpage is simple, just get the id of the element you wish to delete and append it at the end of the AJAX url.

PUT /reset Request

URI Endpoint: 'http://localhost:8080/hava/reset'

Error Codes: No Content: 204, Internal Err: 500

HTTP response MIME type: text/xml

This API resets the whole SQL database, thus deleting all lines. Integration into a webpage is simple. Do the AJAX call and everything is handled by the API. This is a 'PUT' operation because it UPDATES the entire database by deleting the 'phones' table and also adding the initial element.

Input/Output Examples:

  • /phone

    • Input: id

    Output: JSON 'phone' object

  • /phones

    • Input: -

    Output: array containing all 'phone' objects in JSON format

  • /post

    • Input: JSON 'phone' object without id attribute

    Output: code 201 (Created)

  • /update

    • Input: JSON 'phone' object WITH id attribute

    Output: code 204 (No Content)

  • /delete

    • Input: id

    Output: code 204 (No Content)

  • /reset

    • Input: -

    Output: code 204 (No Content)
